# Markdown Cheat Sheet (Concise)

## Headings
- # H1  
- ## H2  
- ### H3

## Emphasis
- **Bold**: `**bold**`  
- *Italic*: `*italic*`  
- ***Bold+Italic***: `***both***`

## Lists
- Unordered: `- item` or `* item`  
- Ordered: `1. First` `2. Second`

## Links & Images
- Link: `[text](https://example.com)`  
- Image: `![alt text](image.png)`

## Code
- Inline: `` `code` ``  
- Block (fenced):
```bash
```bash
# shell example
ls -la
```
```

## Blockquote
- `> Quote text`

## Horizontal Rule
- `---` or `***`

## Table
- Basic:

| Name | Type |
|------|------|
| Alice | Admin |

## Task List
- `- [ ]` open  
- `- [x]` done

## Escaping
- Use backslash `\` before special chars: `\*literal*`

## HTML (inline)
- Allowed for advanced formatting: `<kbd>Ctrl</kbd>`

## Tips
- Add two spaces at line end for a line break.  
- Use fenced code blocks with language for syntax highlighting.
